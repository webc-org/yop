# Internationalization

## Philosophy

Components use **English by default**. For translated strings, pass them as props. Translations are handled by the consuming application.

## How It Works

```tsx
// Default English
<InputDate />  // Uses "Select date", "Cancel", "Apply"

// With translations (from consuming app)
<InputDate
  selectDateLabel={t.selectDate}
  cancelLabel={t.cancel}
  applyLabel={t.apply}
  months={t.months}
  daysShort={t.daysShort}
/>
```

## Usage Example

### Next.js with next-intl

```tsx
import { useTranslations } from 'next-intl'
import { InputDate, InputFile, Pagination } from '@pushui/react'

function MyForm() {
  const t = useTranslations('components')

  return (
    <>
      <InputDate
        label="Birth date"
        selectDateLabel={t('date.selectDateLabel')}
        previousMonthLabel={t('date.previousMonthLabel')}
        nextMonthLabel={t('date.nextMonthLabel')}
        cancelLabel={t('date.cancelLabel')}
        applyLabel={t('date.applyLabel')}
        months={t.raw('date.months')}
        daysShort={t.raw('date.daysShort')}
      />

      <InputFile
        label="Documents"
        chooseFileLabel={t('file.chooseFileLabel')}
        chooseFilesLabel={t('file.chooseFilesLabel')}
        noFileSelectedLabel={t('file.noFileSelectedLabel')}
        filesSelectedLabel={(n) => t('file.filesSelectedLabel', { n })}
        dropFileLabel={t('file.dropFileLabel')}
        dropFilesLabel={t('file.dropFilesLabel')}
        multiple
        dropzone
      />

      <Pagination
        currentPage={1}
        totalPages={10}
        aria-label={t('pagination.ariaLabel')}
        firstPageLabel={t('pagination.firstPageLabel')}
        previousPageLabel={t('pagination.previousPageLabel')}
        nextPageLabel={t('pagination.nextPageLabel')}
        lastPageLabel={t('pagination.lastPageLabel')}
        pageLabel={(n) => t('pagination.pageLabel', { n })}
      />
    </>
  )
}
```

## Naming Convention

- `*Label` - Button text, aria-label, short text
- `*Text` - Longer text content
- Arrays (e.g., `months`, `daysShort`) - Ordered lists of strings
- `{n}` - Placeholder for dynamic values (count, page number)
